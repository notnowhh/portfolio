# Jishin A | Personal Portfolio 

Welcome to the source code of my personal portfolio website! This project was developed as my Semester 2 Fundamentals of Computing (FOC) mini-project at FISAT (KTU).

## 🌐 Live Demo
Check out the live website here: **https://notnowhh.github.io/portfolio/**

## 🛠️ Tech Stack
* **HTML5:** Used semantic HTML (`<nav>`, `<main>`, `<section>`) for clean, readable code structure.
* **CSS3:** Built using a custom dark theme, utilizing CSS Variables (`:root`) for color management and Flexbox for precise element alignment. No external frameworks were used.
* **JavaScript:** Implemented vanilla JavaScript for client-side form validation on the Contact page.

##  Key Features
* Modern UI with a frosted-glass floating navigation bar.
* Multi-page architecture (Home, Projects, Contact).
* Custom gradient text and interactive hover states.
* Built-in form validation to ensure required fields are completed before submission.
